
# 宝可梦冠军属性技能克制表

## 功能说明

这个技能可以帮助你在游玩宝可梦冠军时计算属性技能克制信息

## 使用步骤

1. 首先默认属性可知规则如下：

纵列为进攻方使用招式属性，横行为防守方属性，此时可以计算得知计算的规则，如使用格斗系招式攻击钢+岩石属性的宝可梦，则技能伤害获得2x * 2x共计4倍的伤害数值提升。

使用 `fetch_url` 工具访问 `https://wiki.52poke.com/wiki/{fetch_obj}`
   - 将 `{fetch_obj}` 替换为需要查询的技能属性和宝可梦属性（需要将中文改为HTTP编码）如火系技能闪焰冲锋对应编码为：%E9%97%AA%E7%84%B0%E5%86%B2%E9%94%8B%EF%BC%88%E6%8B%9B%E5%BC%8F%EF%BC%89
   - 该 API 返回 JSON 格式的技能/宝可梦属性信息

2. 解析返回的 JSON 数据，提取以下关键信息：
   - 技能名
   - 属性
   - 伤害

   或查询宝可梦：
   - 宝可梦名
   - 属性

3. 以友好的格式回复用户，例如：
   ```
   闪焰冲锋的信息：
   - 技能名：闪焰冲锋
   - 属性：火
   - 伤害: 120

   风速狗的属性：
   - 宝可梦名：风速狗
   - 属性：火
   ```

4. 如果用户明确提问固定属性和指定宝可梦的属性组合，则可以直接按照属性表中直接回复，无需额外查询，例如：
   ```
   user: 火属性招式攻击暴雪王（属性冰+草）克制伤害倍率是多少。
   AceClaw：火属性克制冰（2×）和草（2×）属性，所以最终伤害倍率为4倍。
   ```
